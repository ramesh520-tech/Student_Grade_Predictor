------------------Student Grade Predictor (ML + Flask)---------------------


This project is a Machine Learning web application that predicts a student's grade based on their 
marks in different subjects.

Features:
       1) Predicts grade (A, B, C, Fail) based on marks
       2) Built using Machine Learning model
       3) Simple and clean Flask web interface
       4) Easy to run locally
Technologies Used:
       1) Python
       2) Flask
       3) Scikit-learn
       4) Pandas
       5) Html

       
How It Works:
     1. User enters marks for subjects
     2. Data is sent to Flask backend
     3. Trained ML model predicts the grade
     4. Result is displayed on the webpage


Example Input
      Maths: 90
      English: 85
      Science: 88
      Telugu: 92
      Biology: 89
Output: Grade A
